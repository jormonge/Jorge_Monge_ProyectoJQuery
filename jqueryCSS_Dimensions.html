Dimensions

The width() and height() methods can be used to get and set the width and height of HTML elements.

Let's set both the width and height of a div to 100px, as well as set a background color for it:

For Example 

                                $("div").css("background-color", "red");
 
                                $("div").width(100);
 
                                $("div").height(100);

Further Dimensions methods

The width() and height() methods get and set the dimensions without the padding, borders and margins.

The innerWidth() and innerHeight() methods also include the padding.

innerWidth= padding + width + padding

The outerWidth() and outerHeight() methods include the padding and borders.

outerWidth= border + padding + width + padding + border

[Margin is not included on any of them, so it does not alter its width or height]

By running the code we noticed only the outerWidth() and outerHeight() methods return the actual set value.

But this is only because we are using the CSS border box property
box-sizing: border-box;

Without modifying it, the standard box-sizing method is content-box- in which the actual height and width are provided by the with() and height() methods.
